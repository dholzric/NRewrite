# API Endpoints

## Authentication
- `POST /api/auth/login`
- `POST /api/auth/register`
- `GET /api/auth/me`

## Notes
- `GET /api/notes`
- `POST /api/notes`
- `GET /api/notes/:id`
- `PUT /api/notes/:id`
- `DELETE /api/notes/:id`

## Users
- `GET /api/users`
- `GET /api/users/:id`
- `PUT /api/users/:id`
